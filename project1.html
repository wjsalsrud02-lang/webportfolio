Overview

실제 서점 웹사이트의 구조와 사용자 흐름을 참고하여 제작한 반응형 웹 프로젝트입니다.
메인 페이지와 서브 페이지를 구현하고, 카카오 Open API와 연동하여 도서 정보를 불러올 수 있도록 구성하였습니다.
또한 HTML, CSS를 중심으로 퍼블리싱과 UI 구현 경험을 쌓는 데 중점을 두었으며, 실제 서비스 형태에 가까운 화면을 구성하여 프론트엔드에 집중한 프로젝트입니다.
