Overview

AI 챗봇을 활용한 사용자 맞춤형 국내 여행지 추천 서비스를 구현하기 위해 기획한 프로젝트입니다.
React 기반 프론트엔드와 Flask 백엔드를 RESTful API 구조로 연동하였으며, OpenAI API를 활용하여 사용자와의 대화를 기반으로 여행지를 추천하는 챗봇 기능을 구현하였습니다.
또한 Figma를 활용해 전체 사이트 UI/UX 디자인을 주도적으로 설계하였고, 사용자 페이지(User Page)를 담당하여 마이페이지 및 게시글 기능 구현에 참여하였습니다.
프로젝트 완료 후에는 Docker를 활용하여 프론트엔드와 백엔드를 컨테이너화하고 배포 환경까지 구성하였습니다.
